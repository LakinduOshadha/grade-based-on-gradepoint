# grade-based-on-gradepoint

matlab-script.m displays the grade of a subject, based on the grade point using the following table.

Grade     Grade Point
A          4.0
A-         3.7
B+         3.3
B          3.0
B-         2.7
C+         2.3
C          2.0
C-         1.7
D+         1.3
E          0.0
I          Incomplete
